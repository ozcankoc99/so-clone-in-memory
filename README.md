# so-clone-in-memory

A Full Stack demo App :

- uses an in-memory Ddatabase
- is a very simple StackOverflow clone
- has implementations

  - backend :

    - .NET
    - Java
    - NodeJs

  - frontend
    - Angular
    - Vue
    - React

- features JWT authentication

Demos available are at:

- .NET/React : xyz
- .NET/Angular : abc
